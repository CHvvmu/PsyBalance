# Behavioral Inference

AI should infer patterns only from observable behavior.

## Allowed inference

- response windows
- likely return after silence
- repeated avoidance patterns
- task completion after support
- acknowledgement-only replies

## Not allowed

- diagnosis
- coercive engagement judgments
- hidden emotional state claims
- therapy-like autonomous decisions

## Best evidence sources

- `behavior_events`
- `coach_interventions.metadata`
- `task_activity`
- `messages`
