# Architecture Prompts

Use these prompts when asking an AI to help with architecture work.

## Prompt style

- ask for code that matches the existing schema
- require explicit mention of tradeoffs
- forbid invented tables or RPCs
- require evidence from the current repo

## Example

> Describe the current event-first architecture using only the schema and code in the repo.
> List the exact tables, triggers, and projections involved.
> Do not invent any new infrastructure.
